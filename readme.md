# Instagram Scraper Pro

## Deskripsi
Tools untuk scraping Instagram: download gambar, caption, dan video dari multiple akun.

## Fitur
- ✅ Download gambar dari feed (termasuk carousel/multiple images)
- ✅ Download caption/text dari setiap postingan
- ✅ Download video dari reels (menggunakan yt-dlp)
- ✅ Support multi akun sekaligus
- ✅ Manual login (lebih aman, tidak terdeteksi bot)
- ✅ Progress real-time
